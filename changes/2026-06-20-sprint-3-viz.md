# Sprint 3 改动日志 — 多模态音频可视化

## 新增

### 1. `client/src/Visualizer.tsx` (580 行)

完整的多模态音频可视化模块, 含:

**纯函数工具 (可单测)**:
- `int16ToFloat32` — Int16 → Float32 归一化
- `computeRMS` — Root Mean Square 能量计算
- `detectVoiceActivity` — RMS 阈值法 VAD
- `estimatePitchAutocorrelation` — 自相关基频估计 (60-500Hz)
- `binToHz` / `hzToBin` — FFT bin ↔ Hz 映射

**数据结构**:
- `PitchHistory` — 音高环形缓冲 (保留最近 N 帧)
- `SpectrumRing` — 频谱环形缓冲 (cols × bins)

**AudioVisualizer 类**:
- AnalyserNode 双模式 (MediaStream + 外部 Int16Array)
- rAF 60fps 自驱, 不触发 React re-render
- 4 维度绘制: 频谱热力图 / 音高曲线 / VU 能量条 / VAD 灯
- FPS 计数器
- DPR-aware Canvas (高 DPI 屏幕)

**VisualizerPanel React 组件**:
- 接收 `stream?`, `audioData?`, `active` props
- 4 宫格布局, 响应式
- ARIA 友好 (`aria-live="polite"` on VAD)

### 2. `client/src/__tests__/Visualizer.test.ts` (290 行)

30 个单元测试, 覆盖全部纯函数和数据结构, 全部通过.

### 3. `tests/capture_viz.py` (65 行)

Playwright 截图脚本, 启动静态服务 + Vosk 后端, 点击测试按钮, 1.5s 后截图.

## 修改

### 1. `client/src/App.tsx`

- 新增 import: `VisualizerPanel`
- 新增 state: `latestAudio: Int16Array | null`, `mediaStream: MediaStream | null`
- `handleAudioData` 中: `setLatestAudio(audioData)` — 推给 Visualizer
- `startRecording` 中: `setMediaStream(stream)` — 暴露 MediaStream 给 Visualizer (AnalyserNode 模式)
- `stopRecording` 中: `setMediaStream(null)` — 清理
- `playSampleAudio` 的 `onChunk` 中: `setLatestAudio(chunk)` — fallback 模式推数据
- 在 `<main>` 之后, `<Subtitle>` 之前 mount `<VisualizerPanel>`

### 2. `client/src/AudioCapture.ts`

- 新增 `getMediaStream(): MediaStream | null` — 暴露内部 stream 给 Visualizer

### 3. `client/src/styles.css` (+130 行)

- `.visualizer-panel` — 容器 (背景圆角 12px, padding 20px)
- `.visualizer-grid` — 4 列响应式网格 `2fr 1.2fr 0.4fr 0.6fr`, ≤1024px 折 2 列
- `.viz-tile` — 单个 tile (min-height 180px, 内 padding 10px)
- `.viz-tile-header` / `.viz-label` / `.viz-readout-small` — header
- `.viz-canvas` — 通用 canvas 样式 (flex 1, 背景 #0a0a18)
- `.spectrum-canvas` — `image-rendering: pixelated` (频谱格子感)
- `.vad-center` — VAD 灯居中布局
- `.vad-indicator` — 圆点, 默认灰色 56px
- `.vad-indicator.vad-on` — 绿色 + glow + pulse 动画
- `.vad-indicator.vad-off` — 灰色 + 内阴影
- `@keyframes vad-pulse` — 1.2s 呼吸动画
- `@media (prefers-reduced-motion: reduce)` — 关闭 VAD 动画
- `.vad-legend` / `.dot` — 图例
