## 修改文件

- `client/src/App.tsx`
  - 顶层加 `AccessibilityProvider` 包裹 (`AppWithProviders`)
  - 新增 `AppShell` 子组件: 持有 helpOpen 状态, 监听 `useKeyboardShortcuts`
  - 桥接: AppShell 通过 `window.__voskShortcuts` 把快捷键回调转成自定义事件, App 内部 useEffect 监听并调用 startRecording/stopRecording/clearTranscription/playSampleAudio
  - 所有按钮加 `aria-label` 和 `aria-keyshortcuts`
  - 状态显示、连接状态、转写列表加 `aria-live="polite"`
  - 头部加 `<ThemeSwitcher />`
  - 主内容加 `<a href="#main-content" className="skip-link">` (键盘 Tab 跳过装饰)
  - main 加 `role="main"` 和 `id="main-content"`
  - header 加 `role="banner"`, section 加 `aria-label`

- `client/src/Subtitle.tsx`
  - 加 `role="region"`, `aria-label="实时字幕"`, `aria-live="polite"`, `aria-atomic="false"`

- `client/src/TranscriptionRenderer.tsx`
  - results-list 加 `role="log"` + `aria-live="polite"` + `aria-relevant="additions"` + `aria-label`
  - 每个 result-item 加 `role="listitem"`
  - 时间戳加 `aria-hidden="true"` (装饰元素)
  - 延迟标签加 `aria-label="延迟 X 毫秒"`
  - 最终标记加 `aria-label="已确认"`
  - 实时行加 `aria-label="实时识别中"`

- `client/src/styles.css`
  - 变量系统重写, `:root[data-theme="dark"|"light"|"hc"]` 三套配色
  - `[data-theme="hc"]` 用 AAA 7:1 对比度配色
  - `[data-theme="light"]` 高对比度浅色, 模拟 macOS / iOS light mode
  - `@media (prefers-reduced-motion: reduce)` 关闭所有动画 / 过渡
  - `:root[data-motion="reduce"]` JS 控制的减动效开关
  - `.btn:focus-visible`, `.theme-option:focus-visible`, `.monitor-link:focus-visible` 加 2px outline (焦点环)
  - 新增 `.theme-switcher`, `.theme-option`, `.help-overlay`, `.help-dialog`, `.sr-only`, `.skip-link` 样式
